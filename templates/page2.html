ABOUT

This Flask app compares NFC West legends to modern-day stars with a pixel UI theme. The player portraits are pixel-style images stored locally in the project.

Generative AI was used to help plan the UI layout (templates + styling) and to generate the pixel-art portraits used in the app.

Next upgrade ideas: add more stats, add more players per team, or load stats from a JSON file.
